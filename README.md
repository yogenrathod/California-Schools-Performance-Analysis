# Predictors of Academic Excellence 🎓
### California Elementary Schools Data Analysis

**Overview:**
This project identifies why some schools perform better than others using a dataset of 400 California schools.

**Tools Used:**
- **R Studio:** Multiple Linear Regression and custom visualizations.
- **JASP:** ANOVA and T-Testing for rapid statistical validation.

**Key Findings:**
- Our model explains **85%** of school performance variance (R-Squared: 0.848).
- **Teacher Credentials** and **Parent Education** are the strongest actionable drivers for student success.
